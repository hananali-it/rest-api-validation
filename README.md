# REST API Validation with Python

This project demonstrates basic REST API testing and JSON response validation using Python and the requests library.

## Project Structure

- `src/api_test.py` → main API validation script
- `requirements.txt` → project dependencies
- `output_example.txt` → example script output

## Features

- Sends a GET request to a public REST API
- Validates HTTP status code
- Parses JSON response
- Checks for required fields in the response

## Requirements

pip install -r requirements.txt
